# Hotel Management System

##  Description
This is a Java-based Hotel Management System developed using NetBeans. The system is designed to manage hotel operations such as room booking, customer records, and billing.

##  Technologies Used
- Java
- NetBeans IDE
- MySQL database

## Features
- Room booking system
- Customer management
- Check-in and check-out
- Billing system

## My Role
Developed the system as part of an academic project, focusing on backend logic and system functionality.

## Known Issues
- UI can be improved
- Some validation features are incomplete

## How to Run
1. Open project in NetBeans
2. Build the project
3. Run the main class
